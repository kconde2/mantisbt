
Clone the project

```
git clone <this-repository-url-https-or-ssh>
```

To launch project, simply run:

```
make up
```

To stop project, run:

```
make stop
```

To stop and remove all data project, run:

```
make down
```

To navigate into browser, see:

```
[http://localhost:8000](http://localhost:8000)
```
